## **概要**

ゆきねこ｡（ykNe0372）のポートフォリオです。以前ポートフォリオを作成しましたが、ページのほぼ全てを改修することになり、それなら一から作成した方が良いと判断し、新しく公開することにしました。

- 記述する内容毎に、ページを分けて作成しました。なるべく長い距離をスクロールしなくても良くなるようにしました。
